# Drone Detection using YOLOv8

This project implements a real-time drone detection system using a pretrained YOLOv8 model with Python and OpenCV.

## Features
- Real-time object detection
- Uses YOLOv8 pretrained model
- Implemented using Python and OpenCV
- Can detect aerial objects such as drones in images and video streams

## Tech Stack
- Python
- OpenCV
- YOLOv8 (Ultralytics)

## Use Case
This system can be used for:
- Campus security
- Restricted area surveillance
- Aerial threat detection

## Status
Project completed and tested successfully.
